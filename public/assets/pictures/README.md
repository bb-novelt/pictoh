# Pictures / Images

This directory contains test images for the Pict'Oh application.

## Current Test Images

- `arbre.svg` - Tree (arbre)
- `chat.svg` - Cat (chat)
- `eau.svg` - Water (eau)
- `fleur.svg` - Flower (fleur)
- `maison.svg` - House (maison)
- `manger.svg` - Food/Eat (manger)
- `soleil.svg` - Sun (soleil)
- `voiture.svg` - Car (voiture)

## Adding More Pictures

More pictures will be added to this directory in the future. Simply place new image files (SVG, PNG, or JPG) in this directory. The application will automatically detect and load them.

**Note**: The filename (without extension) will be used as the default text associated with the picture.

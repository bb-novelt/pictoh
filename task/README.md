# Task Management

This directory contains the task management system for the Pict'Oh project.

## Directory Structure

- **Todo/** - Contains all pending tasks that need to be completed
- **Done/** - Contains completed tasks with their completion summaries

## For GitHub Copilot Agents

If you are a GitHub Copilot agent working on this project:

1. **Look in the `Todo/` folder** to find the next task to work on
2. **Read the task file** to understand what needs to be done
3. **Complete the task** according to the requirements
4. **Move the task file** from `Todo/` to `Done/` when finished
5. **Append a completion summary** to the file describing what was done

For detailed instructions, see [COPILOT-AGENT-GUIDE.md](../COPILOT-AGENT-GUIDE.md) in the root directory.

## Task Naming Convention

Tasks are named using the format: `X.Y-task-description.md`

- `X` - Epic number
- `Y` - Task number within the epic
- `task-description` - Brief description of the task

## Quick Reference

- **Find next task**: Browse files in `Todo/` folder
- **Complete task**: Move file to `Done/` and append summary
- **Task format**: Each task has a checklist of items to complete
- **Full guide**: See [COPILOT-AGENT-GUIDE.md](../COPILOT-AGENT-GUIDE.md)
